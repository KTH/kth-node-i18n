# Changelog

All notable changes for major version updates will be documented here.

## 2.0.1

### Notable changes

- Added a safe fallback in `message()` when no language messages are loaded in the current module instance
- Added a one-time warning to help diagnose setups with multiple installed copies of `kth-node-i18n`

## 2.0.0

### Breaking changes

- Requires Node.js `>=24`.

### Notable changes

- Updated tooling stack to modern versions (ESLint 9, Jest 30, Husky 9, lint-staged 17).
- Updated package dependencies to current minor and patch versions.
